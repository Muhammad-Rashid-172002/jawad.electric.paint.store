# Jawad Electric, Paint & Sanitary Store Website

This project is a modern and responsive website for **Jawad Electric, Paint & Sanitary Store**.
The website allows customers to browse electrical, paint, and sanitary products and easily contact the store through phone, WhatsApp, or email.

## Features

* Modern and responsive UI
* Product category browsing
* Electrical, Paint, and Sanitary product sections
* WhatsApp contact integration
* Phone call buttons
* Email contact option
* Google Map store location
* Fast and user-friendly interface

## Technologies Used

* HTML
* CSS / Tailwind CSS
* JavaScript
* Node.js

## Run Locally

### Prerequisites

Make sure you have **Node.js** installed on your system.

### Installation Steps

1. Clone the repository

```
git clone https://github.com/Muhammad-Rashid-172002/jawad.electric.paint.store.git
```

2. Navigate to the project folder

```
cd jawad-electric-store
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```

5. Open the app in your browser

```
http://localhost:3000
```
